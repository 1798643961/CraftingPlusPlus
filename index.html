简介

原版模组一般是指在不修改Minecraft游戏本体的前提下，通过命令方块、ooc或数据包等方式对游戏的可玩性做出修改。

本模组原名“刷怪蛋、高压、石头工艺”，自1.3版本起更名为“更多的合成”。目前模组更新至1.8(元素更新)，支持的Minecraft版本为Minecraft Java 1.13。本次更新为该模组最大的一次更新，引入了元素之力，以经验值为能量进行方块处理、工具制作、效果释放、属性附加等，同时在世界生成、交易、合成、烧炼、酿造、种植、称号等方面添加了很多新的内容。

背包合成

所有涉及模组特有物品的合成均采用背包合成，方式如下：将材料放入背包右上角对应的3×3的格子里即可自动得到产物，数量和位置须完全吻合。

技术性提醒

该版本分为19个数据包模块，玩家如需要移除部分功能可将其对应的数据包删除或禁用即可。不过，模块之间有依赖关系，请根据内容保留必要的模块。

本模组命名空间和记分板、标签、组名称前缀为cpp，为便于他人整合使用，烦请避开此前缀。

数据包 cpp_entities

生物修改

  • 僵尸——有更高的几率生成时手持武器或身穿盔甲的几率。
  • 骷髅——有更高的几率生成时身穿盔甲，同时有几率射出药箭。
  • 爬行者——爆炸前的引线时间更短。高压爬行者可天然生成，几率约为5%。
  • 村民——可被手持绿宝石块的玩家吸引。

异变生物

生成生物时有5%几率生成，或在黑暗仪式失败时生成。异变生物和通常的动物外观无差异，但是会主攻靠近并攻击玩家。异变生物会掉落黑暗碎片。

  • 异变牛——对玩家造成凋零和饥饿效果，掉落皮革和腐肉。
  • 异变哞菇——对玩家造成凋零和饥饿效果，掉落红色蘑菇和腐肉。
  • 异变羊——对玩家造成凋零和缓慢效果，掉落黑色羊毛和骨头。
  • 异变猪——靠近后直接爆炸，掉落腐肉和火药。
  • 异变鸡——对玩家造成凋零和中毒效果，掉落羽毛和线。
  • 异变兔子——对玩家造成凋零II效果，掉落小块皮革和骨头。

稀有物无敌

下界之星、不死图腾、龙蛋的掉落物形式将不会被火焰或仙人掌破坏。

破损的刷怪箱

现在破坏刷怪箱可以获得破损的刷怪箱。

附魔之瓶

附魔之瓶将固定掉落4点经验值。玩家在背包合成区域中间放置漏斗而其余8格为空时，会自动将4点经验值转化为附魔之瓶。如果在其上方一格还放置压缩器时，会自动将256点经验值转化为压缩的附魔之瓶。

数据包 cpp_elements

元素之力

通过搭配不同的染料、矿物和生物的稀有掉落物，我们可以合成不同的元素之力。元素之力可通过消耗经验发挥不同的效果，也是参与自动化处理的必需品，同时还可用于合成各种材料和工具。元素之力参与合成时自身不会被消耗。
