本次更新为元素更新，模组版本1.8，支持 Minecraft Java 1.13。本次更新引入了元素之力，增加了经验值在多方面的用途，包括方块处理、饰品制作、状态效果释放和附魔书的制作，同时在原版的世界生成、交易、合成上也添加了很多新内容。

背包合成

涉及模组新增物品的合成均采用背包合成，方式如下：将材料放入背包右上角对应的3×3的格子里即可自动得到产物，数量和位置须完全吻合。

下载与安装

点此下载，点此查看mcbbs介绍帖。

将解压缩后的 datapacks, data, resources.zip 放入 .minecraft/saves/世界名称 文件夹，然后退出世界重进即可。

部分内容支持自定义参数，请打开datapacks/craftingpp/data/cpp/config.mcfunction修改。

机械

压缩机：沿竖直方向依次摆放漏斗、向下的活塞、漏斗，然后将相应物品放入顶部漏斗的第一格即可压缩。支持铁粒、金粒、矿物、粘土球、荧石粉、雪球、西瓜、地狱疣、线、小麦、粘液球、干海草。

制造机：沿竖直方向依次摆放漏斗、工作台、漏斗，然后将相应物品放入顶部漏斗即可。支持造石、造冰雪、造矿物。

粉碎机：沿竖直方向依次摆放漏斗、砂砾、漏斗，然后将相应物品放入顶部漏斗即可。支持碎石、碎矿。

熔炼机：沿竖直方向依次摆放漏斗、岩浆块、漏斗，然后将相应物品放入顶部漏斗，可使用红沙代替沙子。支持炼矿、制盐。

生长机：沿竖直方向依次摆放漏斗、泥土、漏斗，然后将相应物品放入顶部漏斗。支持各种植物。

繁殖机：沿竖直方向依次摆放漏斗、黑曜石、漏斗，然后将相应物品放入顶部漏斗。支持各种怪物。

营养均衡

玩家需要保持适当的体重以避免过于肥胖或过于瘦弱，这可以通过荤素搭配来实现。如果玩家身体超重(Fat>100)会行走缓慢，而体重过轻(Fat<-100)则会变得虚弱。使用测重压力板可以测量出你的体重指数。
